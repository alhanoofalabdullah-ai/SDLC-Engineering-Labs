# Project 15 – DevOps Pipeline Tracker

Enterprise DevOps Pipeline Tracker built using Spring Boot, REST APIs, JPA, MySQL, and DevOps Governance best practices.

---

## Overview

This platform manages CI/CD pipelines across enterprise environments.

Features:

- Pipeline Tracking
- Build Monitoring
- Deployment Tracking
- Jenkins Integration Ready
- GitHub Actions Tracking
- Environment Monitoring
- Release Analytics
- DevOps Governance Dashboard

---

## Business Scenario

Organizations need centralized visibility into:

- Build Status
- Deployment Status
- Environment Health
- Release Progress
- Failed Pipelines
- Production Deployments

This platform helps DevOps teams monitor software delivery.

---

## API Endpoints

GET /api/pipelines

GET /api/pipelines/{id}

GET /api/pipelines/status/{status}

GET /api/pipelines/environment/{environment}

POST /api/pipelines

PUT /api/pipelines/{id}

DELETE /api/pipelines/{id}

PUT /api/pipelines/{id}/deploy
