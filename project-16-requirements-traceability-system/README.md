# Project 16 – Requirements Traceability System

Enterprise Requirements Traceability Platform built using Spring Boot, REST APIs, JPA, MySQL, and SDLC Governance Frameworks.

---

## Overview

This system provides complete end-to-end traceability across the SDLC lifecycle.

Features:

- Requirements Register
- Business Requirements
- Functional Requirements
- Requirements Traceability Matrix (RTM)
- User Story Mapping
- Test Case Mapping
- UAT Mapping
- Release Mapping
- Change Impact Analysis

---

## Business Scenario

Organizations require visibility from:

Business Requirement

↓

Functional Requirement

↓

User Story

↓

Development

↓

Test Case

↓

UAT

↓

Release

This system ensures full requirement coverage and governance.
