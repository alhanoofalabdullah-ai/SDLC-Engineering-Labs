# Project 12 – Software Estimation Tool

Enterprise Software Estimation Tool built using Spring Boot, REST APIs, JPA, MySQL, and Agile Planning methodologies.

---

## Overview

This project helps project managers estimate:

- Story Points
- Development Effort
- Sprint Capacity
- Resource Requirements
- Delivery Duration
- Estimated Cost

---

## Business Scenario

Software projects require accurate forecasting for:

- Resource Planning
- Budget Planning
- Sprint Planning
- Delivery Forecasting
- PMO Reporting

This platform provides centralized estimation capabilities.

---

## API Endpoints

GET /api/estimations

GET /api/estimations/{id}

POST /api/estimations

PUT /api/estimations/{id}

DELETE /api/estimations/{id}

POST /api/estimations/calculate
