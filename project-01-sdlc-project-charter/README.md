# 🚀 Project 01 – SDLC Planning & Project Charter

This project demonstrates the Planning Phase of the Software Development Life Cycle (SDLC).

The objective is to establish a solid foundation before software development begins by defining project goals, scope, stakeholders, risks, budget, timeline, and success criteria.

---

# 📖 Overview

Planning is the first phase of SDLC and one of the most critical activities in successful software delivery.

This project simulates the creation of a complete Project Charter for an enterprise software implementation.

The planning process ensures alignment between business objectives and technical execution.

---

# 🎯 Business Scenario

A company wants to implement an Enterprise Document Management System to improve document control, workflow automation, and collaboration.

The project team must define:

- Project objectives
- Scope
- Stakeholders
- Budget
- Timeline
- Risks
- Success criteria

before development starts.

---

# 📌 Project Objectives

- Define project vision
- Establish business justification
- Identify stakeholders
- Determine project scope
- Create high-level schedule
- Assess project risks
- Define governance structure
