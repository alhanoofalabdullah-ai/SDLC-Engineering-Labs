# Project 04 – Change Request Management System

Enterprise Change Request Management System built using Spring Boot, REST APIs, JPA, MySQL, and SDLC Change Control best practices.

---

# Overview

This project manages software and business change requests.

The system supports:

- Create Change Requests
- Change Impact Assessment
- Change Approval Workflow
- Priority Management
- Change Tracking
- Change Closure
- Audit Trail

---

# Business Scenario

Enterprise projects frequently receive requests for:

- New Features
- Requirement Changes
- Process Updates
- System Enhancements
- Bug Fix Requests

All changes must be controlled through a formal Change Management process.

---

# SDLC Stage

This project focuses on:

- Change Management
- Impact Analysis
- Approval Process
- Release Planning
