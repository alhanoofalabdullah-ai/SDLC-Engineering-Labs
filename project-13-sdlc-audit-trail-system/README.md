# Project 13 – SDLC Audit Trail System

Enterprise Audit Trail Management System built using Spring Boot, REST APIs, JPA, MySQL, and Governance Frameworks.

---

## Overview

This project provides centralized audit tracking for SDLC activities.

Features:

- Audit Logs
- User Activities
- Change History
- Compliance Monitoring
- Security Auditing
- Approval History
- Governance Reporting

---

## Business Scenario

Organizations need complete traceability of:

- User Actions
- Requirement Changes
- Workflow Approvals
- Release Activities
- Risk Updates
- Defect Changes

The system ensures accountability and compliance.

---

## API Endpoints

GET /api/audit-logs

GET /api/audit-logs/{id}

GET /api/audit-logs/user/{username}

GET /api/audit-logs/module/{module}

POST /api/audit-logs

DELETE /api/audit-logs/{id}
