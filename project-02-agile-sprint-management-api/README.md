# Project 02 – Agile Sprint Management API

Enterprise Agile Sprint Management API built using Spring Boot, REST APIs, JPA, MySQL, and Agile SDLC practices.

---

## Overview

This project manages Agile sprint planning and sprint execution.

The system supports:

- Create Sprints
- Update Sprint Details
- Track Sprint Status
- Manage Sprint Goals
- Track Start and End Dates
- Monitor Sprint Velocity
- Filter Sprints by Status

---

## SDLC Stage

This project focuses on:

- Agile Planning
- Sprint Execution
- Iteration Tracking
- Sprint Review
- Delivery Monitoring

---

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/sprints` | Get all sprints |
| GET | `/api/sprints/{id}` | Get sprint by ID |
| GET | `/api/sprints/status/{status}` | Filter sprints by status |
| POST | `/api/sprints` | Create sprint |
| PUT | `/api/sprints/{id}` | Update sprint |
| DELETE | `/api/sprints/{id}` | Delete sprint |
