# Project 19 – Enterprise Change Impact Analysis System

Enterprise Change Impact Analysis Platform built using Spring Boot, REST APIs, JPA, MySQL, and Change Governance best practices.

---

## Overview

This platform evaluates the impact of proposed changes before implementation.

Features:

- Change Impact Analysis
- Dependency Mapping
- Risk Assessment
- Release Impact Analysis
- Requirement Impact Analysis
- UAT Impact Analysis
- Governance Dashboard
- Change Approval Support

---

## Business Scenario

Organizations must understand the impact of changes before approval.

The platform helps assess:

- Business Impact
- Technical Impact
- UAT Impact
- Release Impact
- Risk Exposure
- Dependency Risks

---

## API Endpoints

GET /api/change-impact

GET /api/change-impact/{id}

POST /api/change-impact

PUT /api/change-impact/{id}

DELETE /api/change-impact/{id}

GET /api/dashboard/summary
