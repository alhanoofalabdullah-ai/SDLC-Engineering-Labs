# Project 17 – Software Quality Metrics Dashboard

Enterprise Software Quality Metrics Dashboard built using Spring Boot, REST APIs, JPA, MySQL, and SDLC Governance Frameworks.

---

## Overview

This project centralizes software quality metrics into a single dashboard.

The dashboard aggregates:

- Defects
- Test Cases
- UAT Results
- Releases
- Risks
- Quality KPIs
- Executive Reporting

---

## Dashboard KPIs

### Testing KPIs

- Total Test Cases
- Passed Tests
- Failed Tests
- Test Coverage %

### Defect KPIs

- Open Defects
- Closed Defects
- Critical Defects
- Defect Density

### UAT KPIs

- UAT Passed
- UAT Failed
- Pending Sign-Off

### Release KPIs

- Planned Releases
- Successful Releases
- Failed Releases

### Risk KPIs

- Open Risks
- Critical Risks
- Mitigated Risks
