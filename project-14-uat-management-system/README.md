# Project 14 – UAT Management System

Enterprise User Acceptance Testing (UAT) Management Platform built using Spring Boot, REST APIs, JPA, MySQL, and SDLC Governance practices.

---

## Overview

This project manages User Acceptance Testing activities before production deployment.

Features:

- UAT Planning
- Test Scenarios
- Business Validation
- UAT Execution
- Defect Escalation
- UAT Sign-Off
- Go-Live Readiness
- Executive Reporting

---

## Business Scenario

Before production deployment, business users must validate system functionality.

This platform enables:

- UAT Coordination
- Scenario Tracking
- Business Approval
- Readiness Assessment
- Executive Sign-Off

---

## API Endpoints

GET /api/uat

GET /api/uat/{id}

GET /api/uat/status/{status}

POST /api/uat

PUT /api/uat/{id}

DELETE /api/uat/{id}

PUT /api/uat/{id}/approve

PUT /api/uat/{id}/reject
