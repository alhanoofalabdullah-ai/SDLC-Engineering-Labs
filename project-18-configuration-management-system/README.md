# Project 18 – Configuration Management System

Enterprise Configuration Management Platform built using Spring Boot, REST APIs, JPA, MySQL, and SDLC Governance Frameworks.

---

## Overview

This platform manages software configuration items and baselines throughout the SDLC lifecycle.

Features:

- Configuration Items (CI)
- Baseline Management
- Version Control
- Configuration Audits
- Change Tracking
- Release Mapping
- Configuration Reporting
- CMDB Concepts

---

## Business Scenario

Organizations need centralized control over:

- Software Components
- Infrastructure Assets
- Application Versions
- Release Baselines
- Configuration Audits

This system ensures traceability and governance.
