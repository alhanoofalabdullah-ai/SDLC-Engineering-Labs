# Project 05 – Defect Tracking System

Enterprise Defect Tracking System built using Spring Boot, REST APIs, JPA, MySQL, and QA Governance practices.

---

# Overview

This project manages software defects throughout the testing lifecycle.

The system supports:

- Defect Logging
- Defect Assignment
- Severity Classification
- Priority Classification
- Resolution Tracking
- Defect Closure
- QA Reporting

---

# Business Scenario

Software testing teams need a centralized platform to manage:

- Bugs
- Incidents
- Enhancement Defects
- Testing Results
- Root Cause Analysis

This system helps QA and Development teams track defects from discovery to closure.

---

# SDLC Stage

This project focuses on:

- Testing

# API Endpoints

GET /api/defects

GET /api/defects/severity/{severity}

POST /api/defects

PUT /api/defects/{id}
