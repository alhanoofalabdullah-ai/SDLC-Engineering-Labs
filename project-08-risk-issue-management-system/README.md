# Project 08 – Risk & Issue Management System

Enterprise Risk & Issue Management System built using Spring Boot, REST APIs, JPA, MySQL, and Governance Frameworks.

---

# Overview

This project manages project risks and issues throughout the SDLC lifecycle.

The platform supports:

- Risk Register
- Issue Register
- Risk Assessment Matrix
- Mitigation Plans
- Escalation Workflow
- Risk Ownership
- Governance Reporting

---

# Business Scenario

Organizations must proactively identify and manage:

- Project Risks
- Operational Risks
- Technology Risks
- Project Issues
- Escalations

This system provides centralized risk governance and issue management.

---

# SDLC Stage

This project focuses on:

- Risk Management
- Issue Management
- Governance
- Escalation Management
- Project Controls

# API Endpoints

## Risks

GET /api/risks

GET /api/risks/{id}

GET /api/risks/status/{status}

POST /api/risks

PUT /api/risks/{id}

DELETE /api/risks/{id}

---

## Issues

GET /api/issues

GET /api/issues/{id}

GET /api/issues/status/{status}

POST /api/issues
