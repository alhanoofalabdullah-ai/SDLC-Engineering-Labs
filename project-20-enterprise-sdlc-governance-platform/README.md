# Project 20 – Enterprise SDLC Governance Platform

Enterprise SDLC Governance Platform built using Spring Boot, REST APIs, JPA, MySQL, and Enterprise Governance Frameworks.

---

## Overview

This platform provides complete SDLC governance across the software delivery lifecycle.

Modules:

- Requirements Management
- User Story Management
- Test Management
- UAT Management
- Risk Management
- Release Management
- Audit Management
- Executive Dashboard
