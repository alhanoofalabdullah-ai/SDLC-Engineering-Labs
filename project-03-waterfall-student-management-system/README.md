# Project 03 – Waterfall Student Management System

Student Management System built using Spring Boot, REST APIs, JPA, MySQL, and Waterfall SDLC methodology.

---

## Overview

This project demonstrates a traditional Waterfall SDLC implementation.

The system supports:

- Student Registration
- Student Profile Management
- Department Tracking
- Enrollment Status
- Academic Year Tracking
- CRUD APIs

---

## SDLC Stage

This project focuses on Waterfall phases:

- Requirements
- Design
- Implementation
- Testing
- Deployment
- Maintenance

---

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/students` | Get all students |
| GET | `/api/students/{id}` | Get student by ID |
| GET | `/api/students/department/{department}` | Filter by department |
| POST | `/api/students` | Create student |
| PUT | `/api/students/{id}` | Update student |
| DELETE | `/api/students/{id}` | Delete student |
