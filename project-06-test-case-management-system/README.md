# Project 06 – Test Case Management System

Enterprise Test Case Management System built using Spring Boot, REST APIs, JPA, MySQL, and QA Governance practices.

---

# Overview

This project manages software testing activities and test cases.

The platform supports:

- Test Case Repository
- Test Execution Tracking
- Test Result Recording
- UAT Support
- Test Coverage Monitoring
- QA Metrics
- Test Reporting

---

# Business Scenario

Quality Assurance teams require a centralized platform to:

- Create Test Cases
- Execute Tests
- Track Results
- Manage UAT Activities
- Generate Test Reports

The system ensures testing governance and release readiness.

---

# SDLC Stage

This project focuses on:

- Test Planning
- Test Execution
- UAT Management
- Quality Assurance
