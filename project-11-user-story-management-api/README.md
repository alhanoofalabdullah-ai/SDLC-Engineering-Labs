# Project 11 – User Story Management API

Enterprise User Story Management Platform built using Spring Boot, REST APIs, JPA, MySQL, and Agile Scrum practices.

---

## Overview

This project manages Agile user stories throughout the product development lifecycle.

Features:

- Product Backlog
- User Stories
- Acceptance Criteria
- Story Points
- Sprint Assignment
- Story Status Tracking
- Agile Reporting

---

## Business Scenario

Agile teams require a centralized platform to manage:

- Product Backlogs
- User Stories
- Sprint Planning
- Story Prioritization
- Acceptance Criteria
- Delivery Tracking

---

## API Endpoints

GET /api/user-stories

GET /api/user-stories/{id}

GET /api/user-stories/status/{status}

GET /api/user-stories/sprint/{sprint}

POST /api/user-stories

PUT /api/user-stories/{id}

DELETE /api/user-stories/{id}
